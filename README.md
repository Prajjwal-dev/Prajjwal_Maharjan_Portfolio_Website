# Prajjwal Portfolio (React - Vite)

Development:

```bash
cd react-site
npm install
npm run dev
```

Build:

```bash
npm run build
npm run preview
```

Notes:
- Static assets (images, pdfs) should be placed in the project root `react-site/public` or referenced from the parent folder if serving locally.
- Update `src/data/certificates.js` to add more certificates and fields.
